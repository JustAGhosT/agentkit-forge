## Platform Overview

Supermaven is an AI code completion tool known for its massive 1M-token context
window and extremely fast response times. It focuses on delivering the fastest
completions in the market while maintaining deep local code understanding.

### Key Capabilities

- **1M Token Context Window** — largest in the industry for code understanding
- **Ultra-Fast Completions** — optimized for minimal latency
- **Deep Local Understanding** — indexes your codebase locally
- **Multi-Line Suggestions** — context-aware block completions
- **Multi-IDE** — VS Code, JetBrains, Neovim
- **Team Features** — shared context, collaboration tools
- **Privacy Options** — local processing modes available
