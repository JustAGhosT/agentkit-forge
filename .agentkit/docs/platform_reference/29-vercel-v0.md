## Platform Overview

Vercel v0 specializes in converting prompts and images into production-grade
React (and Tailwind CSS) UI components. Unlike full-stack builders, v0 focuses
exclusively on the frontend layer, producing the cleanest, most robust UI code
for React/Next.js teams.

### Key Capabilities

- **Prompt-to-UI** — generate React components from text descriptions
- **Image-to-UI** — convert screenshots/designs into code
- **Tailwind CSS** — generates Tailwind-styled components
- **shadcn/ui** — outputs components compatible with shadcn/ui library
- **Vercel Deploy** — seamless deployment integration
- **Code Export** — copy/paste or install via CLI
- **Iterative Refinement** — chat-based iteration on generated components

### What AgentKit Forge Generates

v0 is primarily a generative UI tool. AGENTS.md is not directly consumed but
may be useful when integrating generated components into a larger project.

- **React/Next.js frontend teams** — production-grade UI components
- **Design-to-code workflows** — image input to working components
- **shadcn/ui projects** — native compatibility

### Not Ideal For

- **Full-stack development** — no backend, database, or API support
- **Non-React projects** — React/Tailwind only
- **Complex application logic** — UI generation only
- **File-based AI configuration** — no rule/instruction system
