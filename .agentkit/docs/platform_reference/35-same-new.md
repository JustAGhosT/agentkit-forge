## Platform Overview

Same.new is a browser-based AI app builder that generates production-ready
Next.js/React applications from natural language prompts or by cloning
existing websites via URL. It combines prompt-to-app with URL-based cloning
for rapid prototyping.

### Key Capabilities

- **Prompt-to-App** — generate full-stack Next.js apps from descriptions
- **URL Cloning** — paste any public URL to replicate its UI and structure
- **Full-Stack Generation** — React/Next.js frontend + API backend
- **One-Click Deploy** — Netlify deployment integration
- **GitHub Export** — download and sync generated code
- **Remixable Projects** — browse and fork community projects
- **TypeScript/JavaScript** — native support
- **Responsive Output** — mobile-optimized, PWA-ready apps

### Best For

- **Rapid prototyping** — URL clone + prompt iteration is fastest path to MVP
- **Learning from existing sites** — clone and study professional designs
- **Next.js projects** — opinionated but production-ready stack
- **Non-technical founders** — describe your app, get working code

### Not Ideal For

- **Existing codebase work** — greenfield-focused
- **Enterprise requirements** — limited governance and security
- **Non-web projects** — web apps only, no mobile native
- **Heavy usage** — token-based pricing can get expensive
