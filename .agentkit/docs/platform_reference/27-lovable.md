## Platform Overview

Lovable is a natural-language-to-app platform that turns prompts into full,
editable source code with both frontend and backend (React + Supabase). It
targets product managers, designers, and entrepreneurs who want to ship real
apps, not just prototypes.

### Key Capabilities

- **Chat Mode** — describe your app in natural language
- **Visual Builder** — design visually with point-and-click
- **Full-Stack Generation** — React frontend + Supabase backend
- **Instant Deploy** — built-in hosting with custom domain support
- **Real-Time Collaboration** — team editing and feedback
- **Mandatory Security Scan** — automated security checks before publish
- **GitHub Export** — export full source code to repositories
- **Iterative Refinement** — conversational iteration on generated apps

### Best For

- **Product managers and designers** — ship apps without coding
- **Fastest time-to-MVP** — full-stack app from a prompt
- **Non-technical founders** — code ownership with exportable source
- **React/Supabase projects** — opinionated but production-ready stack

### Not Ideal For

- **Complex backend systems** — Supabase-only backend limits flexibility
- **Existing codebase work** — greenfield-focused
- **Enterprise requirements** — limited governance and compliance
- **Deep architecture control** — AI makes architecture decisions
