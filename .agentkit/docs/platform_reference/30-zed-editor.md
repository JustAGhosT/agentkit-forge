## Platform Overview

Zed is a Rust-based, GPU-accelerated code editor built for speed and
real-time collaboration. Its AI integration supports agentic editing,
local Zeta edit prediction, inline refactoring, and multi-model flexibility.
Open-source with an Apache 2.0 license.

### Key Capabilities

- **Agentic Editing** — describe changes and AI drafts diffs for approval
- **Zeta Edit Prediction** — local AI-driven multi-line code prediction
- **Inline Assistant** — highlight code for AI refactoring, docs, explanations
- **Multi-Model Support** — OpenAI, Anthropic, Google, DeepSeek, Mistral, Ollama
- **Agent Client Protocol (ACP)** — external agents can connect to Zed
- **Real-Time Collaboration** — multiplayer coding with or without AI
- **Blazing Fast** — Rust + GPU acceleration for minimal latency
- **Open-Source** — Apache 2.0 license, community-driven
