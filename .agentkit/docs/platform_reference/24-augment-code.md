## Platform Overview

Augment Code is an enterprise-grade AI coding assistant with a deep Context
Engine that understands entire codebases ‚Äî patterns, dependencies, and
architecture. It excels at large-codebase scenarios and integrates with
development workflow tools like Slack, Jira, Linear, and Notion.

### Key Capabilities

- **Context Engine** ‚Äî maintains persistent understanding of entire codebase
- **AI Agent** ‚Äî autonomous feature implementation, bug fixing, refactoring
- **Smart Completions** ‚Äî multi-line, block, context-aware code suggestions
- **Code Review** ‚Äî PR summaries, inline comments, guideline adherence
- **Workflow Integrations** ‚Äî Slack, GitHub, Jira, Linear, Notion, Confluence
- **200K Token Context** ‚Äî handles very large codebases
- **Enterprise Security** ‚Äî SOC 2 Type II, ISO 42001, CMEK, SIEM, SSO
