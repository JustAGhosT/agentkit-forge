## Platform Overview

PearAI is an open-source AI code editor (VS Code fork) that bundles multiple
best-in-class AI tools — Aider, Supermaven, Continue, Mem0, and Perplexity —
into a single unified workspace. It features an AI Router that auto-selects
the best model for each task.

### Key Capabilities

- **AI Router** — auto-selects GPT-4, Claude, LLaMA per task context
- **Bundled AI Tools** — Aider, Supermaven, Continue, Mem0, Perplexity built-in
- **Smart Memory Layer** — self-improving memory of preferences and context
- **Zero Data Retention** — code indexing and memory happen locally
- **AI Chat & Debugging** — embedded conversational assistant
- **Agent Tools** — autonomous feature generation and bug resolution
- **VS Code Compatibility** — full extension support, easy migration
- **Open-Source** — Apache 2.0, community-driven
