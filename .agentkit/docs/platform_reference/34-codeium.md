## Platform Overview

Codeium is the AI engine behind Windsurf IDE, also available as a standalone
IDE extension. It provides fast, multilingual code completions with a generous
free tier. Codeium powers both the Windsurf IDE's Cascade agent and its own
extension-based completions.

### Key Capabilities

- **Fast Code Completion** — real-time, context-aware suggestions
- **70+ Languages** — broad language support
- **Generous Free Tier** — unlimited completions for individuals
- **Multi-IDE Support** — VS Code, JetBrains, Neovim, Vim, Emacs, Eclipse
- **Chat Assistant** — conversational coding help
- **Search** — natural language code search across codebase

> **Note:** For the full Windsurf IDE experience (Cascade agent, Flows,
> Memories), see [04-windsurf-ide.md](./04-windsurf-ide.md).

### Best For

- **Budget-zero developers** — unlimited free completions
- **Multi-IDE users** — widest editor support in the market
- **Teams wanting to trial AI** — no cost barrier to adoption
- **Complement to other agents** — pair with Claude Code or Aider for agentic work

### Not Ideal For

- **Agentic coding** — use Windsurf IDE instead for Cascade agent
- **Deep customization** — minimal rule/instruction support
- **Enterprise compliance** — fewer certifications than Copilot or Amazon Q
