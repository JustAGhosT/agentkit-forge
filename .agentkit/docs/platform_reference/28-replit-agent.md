## Platform Overview

Replit is a cloud IDE with deep AI integration, supporting 50+ languages.
Replit Agent provides autonomous full-stack coding — "describe what you want,
the agent builds it." Especially strong for education, experimentation, rapid
prototyping, and full-stack MVPs.

### Key Capabilities

- **Replit Agent v2** — autonomous full-stack app building from prompts
- **Ghostwriter** — real-time AI code suggestions (Copilot-like)
- **Cloud IDE** — zero-setup browser-based development
- **50+ Languages** — Python, JavaScript, TypeScript, Go, Rust, Java, etc.
- **Instant Deploy** — Replit-hosted with custom domains
- **Real-Time Collaboration** — Google Docs-style multiplayer editing
- **Git Integration** — GitHub sync and export
- **Most Generous Free Tier** — among app builders

### Best For

- **Education and learning** — zero-setup, immediate coding
- **Rapid prototyping** — idea to deployed app quickly
- **Multi-language experimentation** — 50+ languages supported
- **Collaborative coding** — real-time multiplayer editing
- **Budget-conscious users** — most generous free tier

### Not Ideal For

- **Enterprise production code** — not designed for large-scale systems
- **Privacy-sensitive projects** — all code on Replit cloud
- **Complex existing codebases** — better for new projects
- **Deep AI customization** — limited configuration options
